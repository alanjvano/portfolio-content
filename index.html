Building a drone using a kit or custom components is popular hobbyist project. Quadcopters are by far the most common type that can be found as they use the fewest stable number of rotors and thus offer the best efficiency in terms of cost, weight, and power. However other types of drones exist such as biplanes, helicopters, bicopters, tricopters, hexacopters, octocopters, and so on. Each type has its own applications and advantages/disadvantages. Although a hexacopter is heavier, less efficient, and more costly than a quadcopter, it has greater stability and is more robust to failer than a quadcopter (i.e. a hexacopter can operate on 5 propellers should one fail).

The goal of this project is to build a hexacopter from scratch, using the minimum amount of custom drone hardware/software. In particular the hexacopter detailed in this project does not use a custom drone frame, flight controller, or remote. Instead the hexacopter is constructed using cheap materials that could be found in any hardware store, its operation is based off a Raspberry Pi computer board, and it is controlled with a video game controller. By building the hexacopter from scratch we can gain a much better understanding of how the drone works as well as complete customizability over the operation and control.

This project is still in progress.
